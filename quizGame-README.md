# Python Quiz Game 🎯

An interactive quiz game built with Python. Answer questions, track your score, and test your knowledge — all from the terminal.

## Features

- Multiple choice questions
- Score tracking throughout the game
- Instant feedback after each answer
- Clean command-line interface

## Tech Stack

- **Language:** Python
- **Concepts used:** OOP, control flow, functions, data structures

## How to Run

```bash
# Clone the repo
git clone https://github.com/Abd-Alrahman12/quizGame.git
cd quizGame

# Run the game (Python 3 required)
python main.py
```
